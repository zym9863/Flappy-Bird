# Flappy Bird 🐦

一个使用 HTML5 Canvas 实现的经典 Flappy Bird 游戏克隆版本。

## 🎮 游戏预览

游戏包含经典的 Flappy Bird 玩法：控制小鸟穿过管道障碍，获取尽可能高的分数！

## ✨ 功能特点

- 🎨 纯 Canvas 绘制的游戏画面
- 🐤 可爱的黄色小鸟角色，带有翅膀扇动动画
- 🌿 经典的绿色管道障碍物
- 🏙️ 包含云朵和建筑剪影的背景
- 📊 实时分数显示
- 🔄 游戏结束界面与重新开始功能
- 📱 支持桌面和移动设备

## 🎯 如何运行

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 点击"开始游戏"按钮即可开始

无需安装任何依赖，纯前端实现！

## 🕹️ 游戏操作

| 操作方式 | 功能 |
|---------|------|
| `空格键` | 让小鸟向上飞 |
| `↑` 上箭头 | 让小鸟向上飞 |
| 鼠标点击 | 让小鸟向上飞 |
| 触摸屏幕（移动端） | 让小鸟向上飞 |

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 游戏逻辑
- **Canvas API** - 游戏画面渲染

## 🎲 游戏规则

1. 点击屏幕或按空格键让小鸟向上飞
2. 小鸟会因重力自动下落
3. 穿过管道之间的空隙获得分数
4. 碰到管道或地面则游戏结束
5. 挑战你的最高分！

## 🔧 游戏参数

游戏中的关键参数（可在 `script.js` 中调整）：

| 参数 | 值 | 说明 |
|-----|-----|------|
| `GRAVITY` | 0.25 | 重力加速度 |
| `FLAP_SPEED` | -5.5 | 跳跃速度 |
| `PIPE_WIDTH` | 52 | 管道宽度 |
| `PIPE_SPACING` | 140 | 上下管道间距 |
| `PIPE_SPEED` | 2.5 | 管道移动速度 |
| `BIRD_RADIUS` | 12 | 小鸟半径 |

## 📝 开发说明

- 游戏使用面向对象的方式组织代码，包含 `Bird` 和 `Pipe` 两个主要类
- 使用 `requestAnimationFrame` 实现流畅的游戏循环
- 支持碰撞检测，包括小鸟与管道、地面的碰撞
